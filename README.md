🚀 ISMATH Dashboard

A modern and responsive **Admin Dashboard** built using **HTML5** and **CSS3**.

This project was created to strengthen my frontend development skills by designing a professional dashboard interface without using JavaScript frameworks or external libraries.

## 📌 Project Overview

ISMATH Dashboard is a multi-page admin panel that provides a clean and user-friendly interface for managing data and monitoring activities.

The dashboard includes different sections commonly found in real-world admin systems such as:

- Dashboard
- Analytics
- Users
- Orders
- Messages
- Notifications
- Settings
- Login Page
- Sign Up Page
